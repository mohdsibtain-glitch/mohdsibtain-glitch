<summary><b>рџЋ“ &nbsp;Student Management System вЂ” MySQL Database Project</b></summary>
<br/>

> A relational database design covering the full academic lifecycle of a college вЂ” departments, students, courses, enrollments, marks, and attendance вЂ” built to demonstrate SQL from beginner joins through advanced procedural logic.

**Schema:** 7 tables (`departments`, `students`, `courses`, `enrollments`, `marks`, `attendance`, `attendance_alerts`) with full referential integrity via foreign keys.

**Query progression demonstrated**
- **Beginner:** filtered selects, joins, `GROUP BY` aggregations
- **Intermediate:** attendance % with `CASE`-based status flags, top scorer per course, pass/fail grading logic, students with no enrollments
- **Advanced:** correlated subqueries, `RANK()` / `DENSE_RANK()` window functions, multi-CTE summary reports, a reusable `student_performance` **VIEW**, a `GetReportCard` **stored procedure**, and an `AFTER INSERT` **trigger** that auto-flags attendance shortages
- Indexing strategy (`CREATE INDEX` / `UNIQUE INDEX`) for query optimization on frequently filtered columns
